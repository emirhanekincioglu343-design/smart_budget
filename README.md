Smart Budget 💰

A modern and responsive Dynamic Income & Expense Analyzer built with React, TypeScript, and Tailwind CSS. The application helps users track their financial activities, analyze spending habits, and make smarter budgeting decisions through interactive charts and real-time calculations.

🚀 Features
📈 Income & Expense Tracking
Add, edit, and delete transactions easily.
Categorize income and expenses for better organization.
📊 Dynamic Financial Analysis
Automatically calculates total income, expenses, and remaining balance.
Displays monthly and category-based spending insights.
📉 Interactive Charts
Visualize financial data using charts and graphs.
Monitor spending trends over time.
🏷️ Category Management
Organize transactions into categories such as:
Food
Transportation
Entertainment
Housing
Salary
Investments
Other
🔍 Filtering and Search
Filter transactions by category, date, or transaction type.
Quickly search for specific entries.
🌙 Responsive Modern UI
Fully responsive design for desktop, tablet, and mobile devices.
Clean and intuitive user interface built with Tailwind CSS.
⚡ Real-Time Updates
Financial summaries update instantly whenever transactions change.
🛠️ Technologies Used
React – Component-based frontend framework
TypeScript – Type-safe JavaScript development
Tailwind CSS – Utility-first CSS framework
React Hooks – State and lifecycle management
Chart Library (such as Chart.js or Recharts) – Data visualization
